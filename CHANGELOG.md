# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0.0] - 2026-04-04

### Added
- **ライブタイムスタンプメモ**: 録音中にタイムスタンプ付きメモを即座に記録。/d /a /r ショートカットで決定・アクション・リスクを瞬時に分類
- **タイムラインビュー**: 文字起こしとメモが時間軸でインターリーブ表示。検索・音声連動シーク対応
- **AI自動エンティティリンク**: 文字起こし完了後、過去のイシューや会議への参照をGeminiが自動検出してリンク
- **会議タグ自動付与**: 設計変更・コスト・工程・安全など10カテゴリのタグをAIが自動分類
- **クロスミーティングRAG検索**: 「防水仕様について過去の定例で何が決まった？」のような横断質問にソース付き回答
- **建設用語カスタム辞書**: プロジェクト固有の専門用語を登録するとGemini文字起こし精度が向上
- **音声ストリーミング再生**: 会議詳細からタイムスタンプクリックで該当箇所の音声を再生
- **会議自動プロジェクト分類**: finalize後にscope_resolverが自動でプロジェクトを推定
- **会議シリーズ管理**: OAC定例・設計定例などシリーズ名でグループ化
- **アクションアイテム → タスク自動作成**: 議事録からアクションを抽出し既存タスクシステムに統合
- **キャリーフォワード**: 同シリーズの前回会議から未完了タスクを自動表示
- **アジェンダテンプレート**: シリーズごとにアジェンダ + 要約プロンプトを設定可能
- **会議リンクグラフ**: ReactFlow + Dagre で会議間の参照関係をグラフ可視化
- **統一プロジェクトマスター**: 全エンティティの参照ポイントとなるprojects_masterテーブル
- **関連エンティティパネル**: 会議詳細にイシューリンク・関連会議・タグを一覧表示
