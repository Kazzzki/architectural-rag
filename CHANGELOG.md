# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0.0] - 2026-04-04

### Added
- **タスク管理PM業務アップグレード**: 5つのビューモード（今日/ボード/リスト/カレンダー/マップ）、担当者・ラベル・サブタスク・マイルストーン対応
- **AIレポート生成**: タスク進捗から週次/月次ステータスレポートを自動ドラフト
- **議事録→タスク抽出**: 会議の文字起こしからGemini AIがアクションアイテムを検出し、選択的にタスク化
- **テーブルビュー**: スプレッドシート風の一覧表示、ソート・一括操作対応
- **「今日のタスク」ビュー**: 期限超過・今日の期限・進行中タスクをダッシュボード表示
- **URLステート永続化**: ビューモード・フィルタ設定がURL検索パラメータに保持され、リフレッシュ後も復元
- **Evidence Trail**: RAG検索結果に確信度スコアと引用元を表示
- **統一プロジェクトマスター**: 全モジュール横断のプロジェクト管理API
- **会議機能拡張**: アジェンダテンプレート、会議リンクグラフ、ライブメモ、音声再生
- **クロスRAG検索**: 複数ナレッジベース横断検索、カスタム辞書、自動分類
- **AI自動エンティティリンク**: 課題間の関係をAIが自動検出しリンク

### Changed
- タスク管理のpage.tsxを11コンポーネントに分割し保守性を向上
- バックエンドタスクAPIを20以上の新エンドポイントで拡張
