# [[基本設計フェーズ]] における [[プラントエンジニア]] の役割と意思決定

## 🎯 主要な決定事項 (Primary Decisions)
- **一次側・二次側配管の接続点（インターフェース）のミリ単位での確定**
  - 背景・理由: DB方式における責任分界（Scope Gap）を排除するため。バルブ、フランジ、計装信号の物理的位置および支持架台の負担区分を明確にしないと、施工現場での干渉や「未施工範囲」の発生、追加費用の請求に直結するため。
- **特殊ガス（LPG・CO2）滞留防止を考慮したピット構造と換気仕様の確定**
  - 背景・理由: LPG（空気より重い）やCO2の漏洩時の滞留による爆発・酸欠リスクを回避するため。構造設計（ピット深さ・防水）と機械換気設計を連動させ、「殺人設計」とならない安全性を担保する必要がある。
- **天井クレーンの走行精度維持のための建築側許容変位量の設定**
  - 背景・理由: S造特有の熱膨張や荷重による変位がクレーンのスパン偏差に影響するため。単なる「Sグレード鉄骨」の指定に留まらず、レールのレベル精度と柱の倒れに関する管理基準を構造設計者と合意する必要がある。
- **主要機器のアンカーボルト・スリーブ位置の「凍結（Freeze）」**
  - 背景・理由: 杭基礎・構造スラブ打設後の「あと施工アンカー」を原則禁止するため。防水層の破壊や鉄筋干渉を防ぐべく、基本設計末尾で主要機器の固定位置を確定させ、構造図に反映させる。
- **高圧ガス工作物と建築確認申請のダブル申請タイムラインの策定**
  - 背景・理由: 高圧ガスの「設置許可」が建築の「確認済証」より遅れることによる着工遅延を防ぐため。両申請のクリティカルパスを同期させ、法的な整合性を確保する。
- **発注者支給機器（OFCI）の受入検査基準と責任所在の明記**
  - 背景・理由: 機器本体は発注者、据付はDB施工者となる場合の「傷」や「不具合」の責任分界を明確にするため。受入時の検収フローを基本設計書に組み込み、紛争を未然に防ぐ。
- **ガス置場周辺の「不可侵領域（No-go Zone）」の図示**
  - 背景・理由: 高圧ガス保安法および消防法に基づく離隔距離（敷地境界、火気、建物開口部等）を物理的に確保するため。意匠設計による植栽や駐輪場の配置といった後続の変更を禁止する。

## 📚 必要知識・根拠法令 (Required Knowledge)
- **高圧ガス保安法（一般高圧ガス保安規則 / 液化石油ガス保安規則）**
  - 適用ポイント: LPG、O2、CO2の貯蔵量に応じた許可・届出区分の特定、第一種・第二種置場の境界距離、火気との離隔（8m以上等）、障壁の設置基準の策定。
- **労働安全衛生法（クレーン等安全規則）**
  - 適用ポイント: 定格荷重3t以上のクレーン設置届の作成、建築構造物の支持力担保、点検用歩廊・梯子の設置基準の遵守。
- **消防法（危険物政令）および防爆指針**
  - 適用ポイント: LPG等の可燃性ガスに関連する防爆エリア（0区、1区、2区）の設定と、それに応じた電気設備の選定。
- **建築基準法（第22条区域、法第35条等）**
  - 適用ポイント: 屋根・外壁の不燃材料化、特殊建築物としての避難距離、排煙設備、非常用照明の適合性確認。
- **日本建築学会（JASS 6）およびクレーン構造規格**
  - 適用ポイント: S造鉄骨の製作・施工精度がクレーンの走行性能に与える影響の評価、および衝撃係数を含む動荷重の算出。

## ⚠️ 重要調整・リスク事項 (Critical Coordination)
- **調整対象: [[構造設計者]]**
  - **想定リスク**: クレーンの動荷重（加減速時の水平力）や大型機器の偏心荷重による杭基礎の不同沈下、およびクレーンと生産機器の共振。
  - **解決戦略 (PM/CM視点)**: プラント側から機器の「許容振動値」と「動荷重計算書」を提示。基本設計段階で床の固有振動数解析と杭配置の最適化を要求し、計算結果を相互承認する。
- **調整対象: [[意匠・建築設計者]]**
  - **想定リスク**: 将来の機器更新ルートの欠如、および法規制による離隔距離内への他設備（駐輪場・植栽等）の侵入。
  - **解決戦略 (PM/CM視点)**: BIMを用いた「機器搬入・更新シミュレーション」を実施。リムーバブルパネルのサイズやクレーン吊り込み範囲を確定させ、図面上に「不可侵領域」を明示する。
- **調整対象: [[設備（電気・機械）設計者]]**
  - **想定リスク**: 特殊ガス漏洩時の排気不全、および始動電流によるトランス容量不足や防爆配線範囲の不整合。
  - **解決戦略 (PM/CM視点)**: HAZOP（リスクアセスメント）を実施し、緊急遮断弁のインターロックロジックを共有。負荷リストに基づき、デマンド率を考慮した受変電容量を確定させる。
- **調整対象: [[ゼネコン施工担当（DB施工者）]]**
  - **想定リスク**: 屋根閉塞後の機器搬入不能、および先行床仕上げと機器据付アンカー施工の工程衝突。
  - **解決戦略 (PM/CM視点)**: 「建築・プラントラップ工程表」を作成。屋根閉塞タイミングと大型機器吊り込みの前後関係をマイルストーンとして設定し、合意する。

### 1. 基本設計フェーズにおける設計プロセス

プラントエンジニアは、発注者の生産計画を「建築的制約」へと翻訳する作業から開始します。

1.  **要求仕様の定義（Requirement Definition）**:
    *   生産ラインのタクトタイム、稼働率、将来の拡張性の確認。
    *   特殊ガス（CO2, LPG, O2）の使用量、供給圧力、純度の特定。
    *   天井クレーンの定格荷重、揚程、走行範囲、同時使用条件の整理。
2.  **プロセスレイアウト（Plot Plan）の策定**:
    *   生産フローに基づいた機器配置図の作成。
    *   メンテナンススペース、フォークリフト等の動線確保。
3.  **ユーティリティ負荷計算**:
    *   電力（動力・電灯）、給排水、圧縮空気、特殊ガスの最大・平均負荷算出。
4.  **法規制の棚卸し**:
    *   高圧ガス保安法、消防法、安衛法等の適用条項の特定。

### 2. 主要な決定事項（Key Decisions）

基本設計フェーズで以下の項目を確定させないと、後の詳細設計で構造変更等の大きな手戻りが発生します。

*   **クレーン仕様の確定**:
    *   クレーン形式（トップランナー型等）、スパン、走行レール芯、ガーター下有効高さ。
    *   **重要:** 走行・横行時の衝撃係数を含む「動荷重」の構造設計への受け渡し。
*   **特殊ガス供給システムの方式**:

    *   配管ルート（ピット、ラック、または天井吊り）の決定。
*   **床荷重とピット計画**:
    *   重荷重機器の配置と、構造スラブへの要求積載荷重（N/mm2）。
    *   配管ピット、防油堤、機器基礎（防振基礎含む）の有無と位置。
*   **環境条件の決定**:
    *   換気回数、温湿度条件、クリーン度の設定（特殊ガス漏洩対策の機械換気含む）。

### 3. 必要な法令知識

プラントエンジニアが主導して確認すべき法規体系です。

*   **高圧ガス保安法（一般高圧ガス保安規則 / 液化石油ガス保安規則）**:
    *   LPG、O2、CO2の貯蔵量に応じた「許可」または「届出」の区分。
    *   第一種・第二種置場の境界距離、火気との離隔（8m以上等）、障壁（防護壁）の設置基準。
*   **労働安全衛生法（クレーン等安全規則）**:
    *   定格荷重3t以上のクレーン設置における「設置届」の作成。
    *   建築構造物としての支持力の担保と、点検用歩廊・梯子の設置基準。
*   **消防法（危険物政令）**:
    *   LPG（可燃性ガス）に関連する防爆エリアの設定。
    *   指定数量以上の危険物を取り扱う場合の「危険物施設」としての基準（本件が該当するか精査が必要）。
*   **建築基準法（法第22条区域関連）**:
    *   屋根・外壁の不燃材料化。
    *   特殊建築物としての避難距離、排煙設備、非常用照明の適合。

### 4. 他分野との調整事項（Interface Management）

DB方式において、プラントエンジニアが調整の主導権を握るべき項目です。

*   **構造設計との調整**:
    *   **クレーン反力:** 走行クレーンによるブラケットへの水平・垂直荷重の伝達。
    *   **構造スラブ:** 杭基礎・構造スラブに対し、大型機器のアンカー固定位置や振動影響の共有。
    *   **Sグレード鉄骨:** 溶接品質や柱・梁の接合部詳細が、クレーンの走行精度（レールの直進性）に影響するため、許容誤差の合意。
*   **意匠・建築設計との調整**:
    *   **搬入出口:** 大型機器の更新を見据えた開口部（シャッター、リムーバブルパネル）のサイズ。
    *   **特殊ガス置場:** 法22条区域における屋外ガスの延焼ライン回避と、不燃材料による囲いの意匠調整。
*   **電気設備との調整**:
    *   **受変電設備容量:** プラント機器の始動電流を考慮したトランス容量の決定。
    *   **防爆配線:** LPG周辺等の防爆区分に応じた電気工事範囲の特定。
*   **PM/CMとの調整**:
    *   **区分表（Scope Matrix）の作成:** 発注者支給機器と工事会社施工範囲の明確化（特に配管の二次側接続）。
    *   **工程管理:** 建築工事とプラント機器据付のラップ（先行床仕上げ、屋根閉塞前の機器吊り込み等）。

### 5. 本プロジェクト特有の専門的アドバイス

*   **CO2・O2の安全性**: 酸欠防止およびガス漏洩検知器の配置計画を基本設計に盛り込むこと。
*   **S造の振動対策**: 天井クレーン作動時の振動が、精密な生産機器に影響を与えないか、構造解析と連携して剛性を確保すること。

このフェーズでのプラントエンジニアの「決定の質」が、最終的な工場の生産性と安全性を左右します。特に特殊ガスとクレーンという動的・化学的要素が絡むため、リスクアセスメントに基づいた設計が不可欠です。

### 1. 初期案に対する批判的指摘

1.  **「DB方式」における責任分界（Scope Gap）の認識不足**
    *   DBでは「誰がどこまでやるか」のグレーゾーンが最大の紛争の種です。初期案の「区分表の作成」という言葉だけでは甘い。特に「一次側配管と二次側配管の接続点（バルブ、フランジ、計装信号）」の物理的な位置と、その支持架台の負担区分を基本設計でミリ単位で合意しなければ、現場で「付かない」「聞いていない」が頻発します。
2.  **「特殊ガス」と「ピット・床構造」の致命的なコンフリクト**
    *   LPGは空気より重く、CO2も滞留します。初期案で「配管ピット」を挙げていますが、**ガス漏洩時の滞留リスクと換気設計**が構造設計（ピットの深さや防水）と連動していません。ピット内にガスが溜まる設計は、爆発や酸欠事故を招く「殺人設計」になりかねません。
3.  **「S造＋クレーン」による建築精度の過信**
    *   S造は熱膨張や荷重による変位が大きいです。クレーンの走行精度（スパン偏差）を維持するために、プラント側から「建築側の許容変位量」を構造設計に突きつける必要があります。初期案の「Sグレード鉄骨」だけでは不十分で、**「クレーンレールのレベル精度と柱の倒れ」の管理基準**をこの段階で握るべきです。
4.  **「杭基礎」と「動荷重・偏心荷重」の連動不足**
    *   杭基礎において、大型機器やクレーンの動荷重による「不同沈下」は致命的です。機器の静荷重だけでなく、加減速時の水平力や偏心荷重を、杭配置の最適化（構造設計）にフィードバックするプロセスが抜けています。

### 2. 追加すべき具体的アクションリスト（PM/CM要求事項）

#### A. 法規・許認可の「先行逃げ切り」戦略
*   **高圧ガス工作物と建築確認申請のタイムライン同期**: 高圧ガスの「設置許可」が建築の「確認済証」より遅れると、着工できません。この「ダブル申請」のクリティカルパスをプラントエンジニアが作成すること。
*   **消防同意の事前協議**: LPG（可燃性）とO2（助燃性）が混在する場合、消防の判断が厳しくなります。基本設計段階で所轄消防との「事前相談記録」を残し、防爆エリアの範囲を確定させること。

#### B. 構造・土木とのインターフェース強化
*   **アンカーボルト・スリーブ埋設図の早期確定**: 杭基礎・構造スラブの打設後に「あと施工アンカー」を打つことは、防水層の破壊や鉄筋干渉を招くため原則禁止。主要機器のアンカー位置を基本設計末尾で「凍結（Freeze）」するプロセスを追加せよ。
*   **振動解析の実施要求**: クレーンと生産機器の共振リスクを評価するため、プラント側から機器の「許容振動値」を提示し、構造設計に床の固有振動数計算を要求すること。

#### C. 特殊ガス・ユーティリティの安全性担保
*   **HAZOP（リスクアセスメント）の実施**: 基本設計段階で、ガスの「漏洩」「過圧」「逆流」などの異常事態を想定したHAZOPを実施し、インターロック（緊急遮断弁）のロジックを確定させること。
*   **ガス置場の「離隔距離」の物理的確保**: 敷地境界線や建物開口部からの離隔距離を、図面上に「不可侵領域（No-go Zone）」として明示し、意匠設計が勝手に植栽や駐輪場を置かないよう釘を刺すこと。

#### D. 施工・メンテナンス性の強制
*   **「機器搬入・更新計画書」の作成**: DB施工者が「建て逃げ」しないよう、将来の機器更新ルートを3Dモデル（BIM）等で検証し、天井クレーンを用いた吊り上げシミュレーションを行うこと。
*   **メンテナンスバルブの操作性**: 高所やピット内のバルブについて、キャットウォークや操作ハンドルの延長が必要か、プラントエンジニアが「運用側の視点」で意匠・設備設計を指導すること。

#### E. DB特有のコスト管理
*   **「発注者支給品（OFCI）」の明確化**: 機器本体は発注者、据付はDB施工者とする場合、その「受入検査」の責任所在を明確にすること。傷がついた場合、どちらの責任か？ これを基本設計書に明記せよ。

### 3. PM/CMからの最終警告

プラントエンジニアは「機械の配置図を書く人」ではありません。**「建築・構造・設備・法規のすべてのベクトルを、生産プロセスという中心点に強制的に向けさせる指揮官」**であるべきです。

特に**「特殊ガス」と「クレーン」は、一歩間違えれば重大事故に直結します。** 上記の指摘事項を設計プロセスに組み込み、「検討中」という言葉を排除し、「いつまでに、誰が、どの数値を確定させるか」の**マイルストーンを再構築**してください。
