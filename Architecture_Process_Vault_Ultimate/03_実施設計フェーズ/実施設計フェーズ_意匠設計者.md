# [[実施設計フェーズ]] における [[意匠設計者（チーフアーキテクト）]] の役割と意思決定

## 🎯 主要な決定事項 (Primary Decisions)
- **高圧ガス保安距離の確定と障壁（防護壁）の配置・構造設計**
  - 背景・理由: 高圧ガス保安法に基づき、LPG・O2等の貯蔵所から火気使用場所や敷地境界線までの保安距離をミリ単位で確定させるため。距離が確保できない場合に設置する「障壁」の構造計算（構造側への依頼）と、意匠的な納まり、メンテナンス動線への影響を基本設計への逆戻りが発生しないタイミングで決着させる必要がある。
- **ガス漏洩検知時の防災連動フロー（換気・シャッター・電源遮断）の定義**
  - 背景・理由: ガス漏洩時に「どの範囲の電源を遮断するか」「どのシャッターを強制開放し、どの換気扇を強運転するか」というフェイルセーフの論理を確定させるため。これは建具仕様（防火シャッターか防爆仕様か）や電気・機械設備設計に直結し、人命に関わる致命的な設計要素となる。
- **クレーン走行レールのレベル精度（±3mm以内）を担保する建屋剛性とクリアランスの規定**
  - 背景・理由: S造の熱膨張や荷重によるたわみがクレーン走行に与える影響を最小化するため。クレーンサドルと柱（耐火被覆厚含む）、梁、照明器具、配管とのクリアランスをミリ単位で規定し、施工時の誤差を吸収する「ふかし代」を明文化する必要がある。
- **将来のクレーン増設および設備更新シナリオの図面化**
  - 背景・理由: 将来的にクレーンが2台連なる、あるいはガス配管が増設される際の柱への負担増や走行範囲の変化をあらかじめ想定し、実施設計図に「将来対応範囲」として明記することで、将来の改修コスト増大と構造的リスクを回避するため。
- **特殊ガス配管ピットの防水・勾配・強制換気および防爆仕様の確定**
  - 背景・理由: ピット内にガスが滞留するリスクを排除するため。ピット底の勾配、集水桝の配置、ピット蓋の防爆仕様、およびピット内強制換気設備の配置を意匠図に反映し、床構造としての安全性を担保する必要がある。
- **労働安全衛生法に適合する点検用歩廊（キャットウォーク）の詳細仕様**
  - 背景・理由: クレーン点検用の歩廊が建築基準法上の「床」か設備の一部かを整理し、手摺の高さ、つま先板の有無、避難階段への接続ルートを労基署の指摘前に確定させるため。避難動線とクレーン稼働域の干渉をゼロにする。
- **DB方式におけるVE提案の判定マトリクス策定**
  - 背景・理由: 施工部門からのコストダウン案（外壁材変更等）が、ガスの保安距離、防爆性能、遮音性能、クレーン走行精度に悪影響を与えないか、意匠設計者が「防波堤」として客観的に判断する基準を設けるため。

## 📚 必要知識・根拠法令 (Required Knowledge)
- **高圧ガス保安法（一般高圧ガス保安規則、液化石油ガス保安規則）**
  - 適用ポイント: 第一種・第二種貯蔵所の届出基準、保安距離の算定、障壁の構造基準、警戒標識の掲示義務の適用。
- **建築基準法 第2条、第22条、第35条（避難規定）**
  - 適用ポイント: 特殊建築物（工場）の用途規制、延焼のおそれのある部分の防火構造、歩行距離・重複歩行距離の算定、およびクレーン歩廊の床面積算入可否の判断。
- **労働安全衛生法（クレーン等安全規則 第21条、第22条）**
  - 適用ポイント: クレーン設置報告書の作成、点検用足場の設置基準、安全通路の確保、および落成検査時の指摘事項（手摺・つま先板等）の事前回避。
- **消防法（危険物政令、火災予防条例）**
  - 適用ポイント: LPG等の危険物貯蔵所における消火設備の選定（移動式粉末消火設備等）、自動火災報知設備の感知器配置、排煙設備の有効性確認。
- **日本建築学会（JSSC）鋼構造建方精度規格**
  - 適用ポイント: Sグレード鉄骨の製作・建方精度と、クレーンメーカーが要求する走行レール精度の許容誤差の整合。

## ⚠️ 重要調整・リスク事項 (Critical Coordination)
- **調整対象: [[構造設計者]]**
  - **想定リスク**: クレーン重荷重を支える杭基礎・地中梁と、特殊ガス配管ピット・排水ピットが物理的に干渉し、現場で配管が通らない事態。
  - **解決戦略 (PM/CM視点)**: 実施設計初期に「床下総合プロット図（1/20）」を作成。杭頭補強範囲とピットの離隔を全箇所チェックし、干渉がある場合は構造スラブの補強方法を合意する。
- **調整対象: [[設備設計者（MEP）]]**
  - **想定リスク**: ガス漏洩検知器の配置が換気死角に設置される、あるいは検知器と連動するシャッターの電源系統がガス遮断範囲に含まれてしまい作動しないリスク。
  - **解決戦略 (PM/CM視点)**: 「特殊ガス・防災マトリクス」を共同作成。ガス種別の比重（上昇・沈下）に基づいた検知器位置と、連動する電気・機械設備の動作シーケンスをフロー図で合意する。
- **調整対象: [[施工部門（DB担当）]]**
  - **想定リスク**: 施工側が工期短縮のために提案する「外壁パネルの規格化」や「鉄骨の簡略化」が、クレーンの走行精度（剛性不足）やガスの保安距離（耐火性能不足）を損なうリスク。
  - **解決戦略 (PM/CM視点)**: 「性能担保項目リスト」を提示。VE提案時には、その変更が法規・クレーン精度・安全性に与える影響を数値で示す比較表の提出を義務付ける。
- **調整対象: [[クレーンメーカー]]**
  - **想定リスク**: 給電レール（トロリー線）の位置が、意匠的な天井仕上げや照明器具のメンテナンス用昇降装置と干渉し、点検が不可能になるリスク。
  - **解決戦略 (PM/CM視点)**: 「クレーン・インターフェース詳細図（1/10）」を作成。柱・梁・耐火被覆・サドル・給電線・照明の全てのクリアランスをプロットし、三者（意匠・設備・メーカー）で承認印を交わす。

### 5. 本プロジェクトにおけるシニアアーキテクトの視点

本件は「S造平屋の工場」という一見シンプルな構成ですが、**「クレーン」と「特殊ガス」という動的・危険的要素**が混在しています。

1.  **「逃げ」の設計**: Sグレード鉄骨は精度が高いものの、クレーンレールのレベル（水平度）調整や、ガス配管の貫通部には一定の余裕（クリアランス）を持たせた詳細設計が必要です。
2.  **将来の更新性**: 工場は将来のライン変更が想定されます。ガス配管の増設スペースや、床の耐荷重に余裕を持たせる提案をPM/CMを通じて発注者に行うことが、長期的な資産価値に繋がります。
3.  **安全設計（フェイルセーフ）**: ガス漏洩時の排気動線と避難動線が交差しないか、クレーン稼働範囲内に立ち入り禁止区域を明確化できるかなど、運用面まで踏み込んだ意匠計画が求められます。

### 1. 「高圧ガス保安法」と「建築基準法」の致命的な不整合リスク
初期案では「事前協議」とさらっと書かれていますが、特殊ガス（特にLPG、O2）は**「第一種置場」等の法規制により、火気使用場所や境界線からの「保安距離」が厳格に定められています。**

*   **指摘事項:**
    *   **保安距離の確定タイミング:** 実施設計の終盤で距離が足りないことが判明すれば、配置計画（建築物の位置）の変更、つまり「基本設計への逆戻り」が発生します。
    *   **防護壁の設計:** 保安距離が確保できない場合の「障壁（防護壁）」の構造計算と意匠的処理が抜けています。
    *   **ガス検知器と連動した換気・遮断:** 漏洩時に「どの範囲の電源を遮断するか」「どのシャッターを強制開放するか」という防災連動フローが意匠図（建具仕様）に反映されていません。

### 2. クレーン走行精度と「建屋の剛性」の認識不足
「クリアランス確保」という表現は甘いです。クレーンが動く工場において、意匠設計者が最も警戒すべきは**「建屋のたわみとクレーンレールの許容誤差」**です。

*   **指摘事項:**
    *   **走行レールのレベル精度:** S造は熱膨張や荷重で動きます。クレーンメーカーが求める「レール芯間誤差±3mm以内」などのシビアな精度を、どのタイミングで誰が担保するのか。
    *   **ブラケットの出幅と柱の有効寸法:** 柱の耐火被覆の厚み、およびクレーンサドルと柱のクリアランスは、ミリ単位の検討が必要です。
    *   **将来のクレーン増設:** 1台の予定が将来2台連なる場合、柱への負担と走行範囲が劇的に変わります。この「将来対応の有無」を確定させ、図面に明記しなければなりません。

### 3. 「床」に関する設計の甘さ（ピットと配管）
工場設計において、床は単なる「仕上げ」ではなく「装置」の一部です。

*   **指摘事項:**
    *   **ガス配管ピットの防水と換気:** 特殊ガス配管をピットに通す場合、ガスが滞留するリスクがあります。ピット内の「勾配」や「強制換気」、およびピット蓋の「防爆仕様」の検討が抜けています。
    *   **杭頭補強とピットの干渉:** 杭基礎である以上、重荷重がかかるクレーン脚部付近の杭と、ガス配管ピットが干渉するリスクが極めて高いです。
    *   **床の不連続点（誘発目地）:** クレーン走行による振動と、特殊ガス設備の精密さを守るための「縁切り」の検討が必要です。

### 4. DB（設計施工一括）方式における「責任境界」の曖昧さ
DB方式では、施工部門が「安く、作りやすい」方向に設計を誘導します。意匠設計者が「シニアアーキテクト」を名乗るなら、その防波堤にならなければなりません。

*   **指摘事項:**
    *   **施工誤差の吸収（逃げ）の明文化:** 「逃げの設計」と抽象的に書かず、具体的に「柱芯から壁面までのふかし代」や「天井内設備の有効高さ」を数値で規定し、施工部門に遵守させる「管理図」を作成すべきです。
    *   **VE提案の判定基準:** 施工側からの「外壁材の変更」等が、ガスの保安距離や防爆性能、あるいはクレーンの騒音対策（遮音）に悪影響を与えないか、判定マトリクスを作成してください。

### 5. 労働安全衛生法（クレーン明細書）への対応
クレーン設置には「落成検査」が必要です。

*   **指摘事項:**
    *   **点検用歩廊（キャットウォーク）の法適合:** 建築基準法上の「床」とみなされるか、設備の一部（工作物）とするかの整理。手摺の高さ、つま先板の有無など、労基署の指摘を受けやすい箇所の詳細図化。
    *   **避難階段との関係:** クレーン点検中に火災が発生した場合の、歩廊からの避難ルートの確保。
