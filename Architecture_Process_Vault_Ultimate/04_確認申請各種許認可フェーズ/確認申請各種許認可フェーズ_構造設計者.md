# [[確認申請・各種許認可フェーズ]] における [[構造設計者]] の役割と意思決定

## 🎯 主要な決定事項 (Primary Decisions)
- **構造設計基準書（Structural Design Criteria）の早期策定とフリーズ**
  - 背景・理由: DB方式において、鉄骨製作図（工作図）の発注や杭の先行発注（ロングリードアイテム）を行うためのデッドラインを明確にするため。荷重条件、採用鋼材、許容たわみ、将来増設マージンを網羅し、これ以降の変更が追加費用・工期遅延に直結することをPM/CM・施工部門と合意する必要がある。
- **高圧ガス防護壁の構造種別および建物フレームとの縁切り判定**
  - 背景・理由: 高圧ガス保安法に基づく防護壁（RC造等）が建築基準法上の「建築物」や「工作物」と重複定義されるリスクを回避するため。万が一の爆発時の受圧を建物本体の構造フレームに負担させるのか、独立した構造体とするのかを早期に決定し、基礎形式やルート判定への影響を確定させる。
- **クレーンランウェイの温度応力対策および走行精度維持機構**
  - 背景・理由: S造平屋の長大な桁行方向における温度変化による伸縮が、クレーンレールの「ハの字」変形を招くリスクがある。たわみ制限の計算だけでなく、長穴処理や調整金物による施工誤差吸収策、および構造エキスパンションジョイントの要否を決定する。
- **構造スラブと土間コンクリートの境界設定および不同沈下対策**
  - 背景・理由: 杭基礎を採用する場合、構造スラブと土間の境界で必ず生じる相対変位を特定するため。特に特殊ガス配管がこの境界を跨ぐ場合、配管破断リスクを回避するためのフレキシブルジョイント設置位置と変位量の算定が不可欠である。
- **クレーンストッパー衝突荷重の伝達経路設計**
  - 背景・理由: クレーンがエンドストッパーに衝突した際の水平力を、どのブレースやフレームで負担するかを明確にする。衝突エネルギーによるブレースの座屈や接合部の破断を防止するための詳細設計を行い、施工部門と製作仕様を合意する。
- **アンカーボルトの先行埋込精度とベースプレート仕様**
  - 背景・理由: Sグレード鉄骨かつクレーン付き物件では、アンカーボルトの設置精度が建方精度に直結する。ノンシュリンクモルタルの厚みや、柱脚の固定度を確保するための詳細仕様を施工部門と早期に合意し、先行発注に備える。
- **屋根・外壁の負圧（吹き上げ）に対する緊結詳細**
  - 背景・理由: 平屋工場は受風面積が大きく、台風時の負圧による屋根飛散リスクが高い。告示1454号に基づくE値・風圧力算定を行い、折板屋根のボルト本数やケラバ（妻側）のサブメンバー補強を構造計算書に明記し、意匠・施工部門と整合させる。

## 📚 必要知識・根拠法令 (Required Knowledge)
- **建築基準法 第20条（構造耐力）および構造計算適合性判定**
  - 適用ポイント: ルート3（保有水平耐力計算）の適用要否の判断。適判機関との事前相談をクリティカルパスに設定し、指摘による断面変更が発注済みの鉄骨に波及するリスクを管理する。
- **クレーン等安全規則（労働安全衛生法）**
  - 適用ポイント: 建築基準法とは異なる「衝撃係数」や「許容応力度」の適用。建築確認とクレーン設置届（労基署）の双方を満足する、より厳しい側の数値を採用した安全側設計基準の策定。
- **高圧ガス保安法（貯蔵所・処理場の構造規定）**
  - 適用ポイント: 特殊ガス（CO2, LPG, O2）の警戒標、防護壁の耐震・耐衝撃設計。特にLPG等の重いガスに対応するための床下ピット深さと、地中梁・杭頭との干渉回避設計。
- **建設省告示第1454号（E値・風圧力）**
  - 適用ポイント: 平屋工場の屋根吹き上げ力算定。折板屋根の緊結部、母屋、および外装材取付下地の強度確認への適用。
- **JSSC/AIJ 鋼構造設計規準（Sグレード鉄骨関連）**
  - 適用ポイント: 高強度鋼材や特殊接合部の特性を反映した計算モデルの構築。溶接詳細および非破壊検査基準の構造図への明記。

## ⚠️ 重要調整・リスク事項 (Critical Coordination)
- **調整対象: [[施工部門（鉄骨製作・建方担当）]]**
  - **想定リスク**: 適判の指摘による断面変更が、既にミリ出し・発注済みの鉄骨製作に与える手戻りリスク。
  - **解決戦略 (PM/CM視点)**: 適判機関との「事前相談」完了日を鉄骨発注のデッドラインとして工程表に組み込み、断面確定後の変更を禁止する「設計凍結合意書」を締結する。
- **調整対象: [[設備設計者（特殊ガス設備担当）]]**
  - **想定リスク**: 建物と土間の相対変位によるガス配管の破断、および深いピットによる基礎梁貫通・杭頭干渉。
  - **解決戦略 (PM/CM視点)**: 構造設計者が算出した「想定変位量データ」を設備側に提供し、フレキシブルジョイントの位置をプロットした「インターフェース図」を作成。確認申請前に三者で検図・捺印を行う。
- **調整対象: [[PM/CM（発注者代理）]]**
  - **想定リスク**: 将来のクレーン容量アップや設備更新に伴う床荷重増が、設計確定後に要求されることによる大幅な手戻り。
  - **解決戦略 (PM/CM視点)**: 「構造設計基準書」において将来マージンの範囲（例：クレーン+5tまで、床荷重+10kN/m2まで等）を数値で明文化し、コスト影響を含めて早期に承認を得る。
- **調整対象: [[行政・指定確認検査機関・労基署]]**
  - **想定リスク**: 建築確認、クレーン設置届、高圧ガス設置許可の三者間で、構造計算の前提条件（積載荷重や衝撃係数）に不整合が生じ、許可が下りないリスク。
  - **解決戦略 (PM/CM視点)**: 各行政窓口への「並行事前協議」を実施。協議記録を全関係者で共有し、一方の変更が他方に波及しないようタイムラインを統合管理する。

### 1. 構造設計プロセス（確認申請・許認可フェーズ）

このフェーズでは、単なる計算だけでなく、行政や審査機関（指定確認検査機関・指定構造計算適合性判定機関）との事前協議が中心となります。

    *   **特殊ガス設備荷重:** ガスボンベ庫やタンク（CO2, LPG, O2）の自重、およびそれらを支持する基礎・架台の地震力を算定。
2.  **構造計算（一貫計算および部材詳細計算）**
    *   平屋のため、地震力よりも「風圧力」や「クレーン動荷重」が部材断面を決定する支配要因になることが多いため、これらの検討を優先します。
    *   **Sグレード鉄骨の反映:** 高強度鋼材や特殊な接合部詳細を用いる場合、その特性を計算モデルに反映。
3.  **基礎・地盤の最終判定**
    *   ボーリング調査結果に基づき、杭種（既製コンクリート杭、鋼管杭等）と支持層を確定。クレーンの振動や重量機器による不同沈下リスクを評価します。
4.  **構造計算適合性判定（適判）への対応**
    *   本規模・用途では、ルート3（保有水平耐力計算）を用いる場合が多く、適判が必須となります。指摘事項を想定した計算書の整理を行います。

### 2. 主要な決定事項（構造設計者の判断）

1.  **クレーンガーダーと柱の接合形式**
    *   ブラケット形式か、段付き柱形式かの決定。クレーンの走行精度を維持するための「剛性」確保と、鉄骨製作コスト（Sグレード）のバランスを判断します。
2.  **構造スラブの仕様**
    *   特殊ガス設備や将来の設備更新に備え、土間コンクリートではなく「構造スラブ（配筋・厚みの確保）」とする範囲の決定。特にLPG等の重量物エリアのひび割れ抑制対策。
3.  **ブレース配置と開口部の整合**

4.  **杭頭接合部および基礎形式**
    *   引き抜き力が生じる場合の杭頭補強、およびクレーン動荷重による基礎の疲労・振動対策の要否。

### 3. 必要な法令知識

1.  **建築基準法 第20条（構造耐力）**
    *   平屋であっても、スパンや高さにより「構造計算適合性判定」の対象となるかどうかの確認。
2.  **建築基準法 第22条（屋根・外壁の制限）**
    *   構造設計としては、不燃材料の選定に伴う屋根自重の算定、および法22条区域特有の外装材取付下地の強度確認。
3.  **クレーン等安全規則（労働安全衛生法）**
    *   建築基準法とは別に、クレーン設置届（労働基準監督署）に必要な構造強度計算（クレーンサドル、ランウェイガーダーのたわみ制限等）の準拠。
4.  **高圧ガス保安法**
    *   特殊ガス（CO2, LPG, O2）の貯蔵所・処理場に係る「警戒標」や「障壁（防護壁）」の構造規定。特に防護壁をRC造とする場合の耐震・耐衝撃設計。
5.  **告示1454号（E値・風圧力）**
    *   平屋工場は受風面積が大きく、屋根の吹き上げ力が大きいため、折板屋根の緊結部や母屋の検討において重要。

### 4. 他分野との調整事項（インターフェース管理）

設計施工一括（DB）方式では、施工部門やPM/CMとの早期合意形成が鍵となります。

*   **意匠設計者との調整:**
    *   **有効高さの確保:** クレーン走行空間、フック下高さと、梁成（はりせい）の干渉チェック。
    *   **ガス排気ルート:** 特殊ガス漏洩時の換気ダクト貫通部による梁貫通制限の確認。
*   **設備設計者との調整:**
    *   **ガス配管支持:** 大口径配管や高圧配管の支持ピッチと、鉄骨小梁の位置調整。
    *   **スリーブ位置:** 構造スラブ内の埋設配管・ピットと、基礎梁・杭の干渉回避。
*   **施工部門（Sグレード鉄骨製作）との調整:**
    *   **製作・建方計画:** Sグレード鉄骨の溶接詳細、非破壊検査基準の確認。
    *   **工期管理:** 杭工事および鉄骨製作のロングリードアイテム（先行発注）に向けた、早期の断面確定。
*   **PM/CM（発注者側）との調整:**
    *   **将来増設計画:** 将来のクレーン容量アップや床荷重増の有無を確認し、構造的な余裕度（マージン）をコストと天秤にかけて合意を得る。
    *   **コスト最適化:** Sグレード採用によるコスト増を、部材の標準化やスパン割の最適化でどう吸収するかを提示。

### 5. シニアアーキテクトとしての視点（リスク管理）

*   **振動問題:** クレーン走行による微振動が、特殊ガス設備の精密センサーや制御盤に悪影響を与えないか、剛性設計の観点から助言します。
*   **防錆・防食:** ガス設備周辺の腐食環境を確認し、鉄骨の塗装仕様（重防食塗装等）を構造設計の前提条件に組み込みます。

### 1. 「初期案」への批判的指摘（甘い認識とリスク）

1.  **DB方式における「確定タイミング」の欠如**
    *   DB方式では、確認申請を出す前に「鉄骨製作図（工作図）の発注」や「杭の先行発注」を行う必要があります。初期案には、どのタイミングで構造断面を「凍結（フリーズ）」させるかの戦略がありません。
2.  **特殊ガス設備（高圧ガス保安法）の過小評価**
    *   高圧ガス保安法に基づく「防護壁」や「貯蔵所」は、建築基準法上の「建築物」や「工作物」と重複して定義される場合があり、構造計算の前提（ルート判定）を根底から覆すリスクがあります。
3.  **クレーンランウェイの「施工誤差」と「剛性」の不一致**
    *   「たわみ制限」の計算だけでは不十分です。S造平屋は温度変化による伸縮が大きく、クレーンの走行レールが「ハの字」に開く、あるいは閉じるリスクがあります。これに対する構造的な吸収策（長穴処理や調整金物）の検討が抜けています。
4.  **床（土間・構造スラブ）の境界条件の曖昧さ**
    *   「構造スラブとする範囲の決定」とありますが、杭基礎の場合、土間と構造スラブの境界で必ず「不同沈下による段差」が生じます。特殊ガスの配管がその境界を跨ぐ場合、破断リスクに直結します。

### 2. 追加・修正すべき具体的実務リスト

#### 【A. 法規・許認可のコンフリクト管理】
*   **高圧ガス防護壁の構造種別確定:**
    *   防護壁をRC造にする場合、その自重による基礎への影響だけでなく、万が一の爆発時の受圧を構造フレームに伝えるのか、独立させるのかを早期決定すること。
*   **クレーン設置届（労基署）との整合性:**
    *   建築確認（指定確認検査機関）とクレーン設置届（労基署）では、計算に用いる「衝撃係数」や「許容応力度」の考え方が異なる。厳しい方の数値を採用する「安全側設計」の基準を明文化し、発注者の承認を得ること。

#### 【B. 構造設計の詳細（トラブル未然防止）】
*   **温度応力と伸縮目地の検討:**
    *   平屋で桁行が長い場合、温度変化による鉄骨の伸縮がクレーン走行に与える影響を計算し、必要に応じて「構造エキスパンションジョイント」または「柱の剛性調整」を行うこと。
*   **クレーンストッパーの衝突荷重:**
    *   クレーンがエンドストッパーに衝突した際の水平力を、どのフレームで負担するか。ブレースの座屈や接合部の破断を招かないための詳細設計。
*   **アンカーボルトの先行埋込精度:**
    *   Sグレード鉄骨かつクレーン付きの場合、アンカーボルトの設置精度が建方精度を左右する。ベースプレートの仕様（ノンシュリンクモルタル厚み等）を施工部門と早期合意すること。

#### 【C. インターフェース管理（他分野との取り合い）】
*   **特殊ガス配管の「フレキシブルジョイント」位置の指定:**
    *   構造設計者が、建物と土間の相対変位量を算出し、設備設計者に「ジョイントが必要な箇所と変位量」をデータとして渡すプロセスを追加すること。
*   **床下ピット・トレンチの構造集約:**
    *   特殊ガス配管や排水のためのピットが基礎梁を貫通する場合の補強、および杭頭との干渉回避。特にLPG（重いガス）のためのピットは深くなる傾向があるため、地中梁設計への影響が甚大。
*   **屋根・外壁の「負圧」対策:**
    *   平屋工場は台風時の「吹き上げ」で屋根が飛ぶリスクが高い。折板屋根のボルト本数や、ケラバ（妻側）のサブメンバー補強を構造計算書に明記すること。

#### 【D. DB方式特有のリスク管理】
*   **「構造設計基準書」の早期作成と合意:**
    *   確認申請前に、荷重条件、採用鋼材、許容たわみ、将来増設の有無を網羅した「構造設計基準書」を発注者・施工部門と取り交わす。これ以降の変更は「追加費用」となるデッドラインを明確にする。
*   **鉄骨発注（ミリ出し）に向けたスケジュール逆算:**
    *   適判の指摘による断面変更が、既に発注済みの鉄骨製作に与える影響を最小化するため、適判機関との「事前相談」の完了日をクリティカルパスに設定すること。

### PM/CMからの最終警告

「シニアアーキテクト」を名乗るのであれば、**「計算が通ること」をゴールにするのではなく、「現場で鉄骨が組み上がり、クレーンがスムーズに走り、ガス漏れが起きない状態を、いかに最短・最安の設計変更なしで実現するか」**に執着してください。

特に、**「土間と構造体の取り合い」**および**「高圧ガス保安法と建築基準法の隙間」**は、最もトラブルが発生しやすいポイントです。ここを構造設計者が主導して調整しない限り、このプロジェクトは必ず停滞します。
