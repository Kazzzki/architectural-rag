# [[確認申請・各種許認可フェーズ]] における [[意匠設計者]] の役割と意思決定

## 🎯 主要な決定事項 (Primary Decisions)
- **防爆エリアの策定と電気設備仕様の確定**
  - 背景・理由: LPG（可燃性ガス）の漏洩滞留リスクに基づき、防爆等級と範囲を確定させる必要がある。これが未確定だと、設備設計側で防爆仕様の照明・スイッチ・モーター選定ができず、実施設計での大幅なコスト増を招くため。
- **CO2漏洩時の酸素欠乏対策を考慮した断面・換気計画**
  - 背景・理由: CO2は空気より重くピット内に滞留し酸素欠乏症を引き起こすリスクがある。床面付近への換気口設置やピット構造の回避など、意匠断面図に安全対策を反映させ、法規上の換気量計算と整合させる必要がある。
- **天井クレーン走行空間およびメンテナンス用付帯施設の配置**
  - 背景・理由: 走行レールの側方に必要な給電用トロリー線スペース、および労働安全衛生法に基づく点検台・タラップの設置場所を確定させる。これらは防火区画の貫通や避難経路に干渉するため、確認申請図への記載が必須である。
- **高圧ガス保安法に基づく法定離隔距離のプロットと配置FIX**
  - 背景・理由: LPG・O2等の貯蔵・処理設備から敷地境界、火気（喫煙所、給湯室排気口等）までの第一種・第二種保安距離を確保する。将来の増設計画を含めた配置図を作成し、行政のガス担当課との事前合意を得るため。
- **床（土間コンクリート）の要求仕様（平坦度・目地割）の確定**
  - 背景・理由: 精密機械設置のための超平坦床（スーパーフラット）の要否や、フォークリフト動線に基づいた目地位置を決定する。不適切な目地配置は床の崩壊を招き、生産設備に致命的な影響を与えるため。
- **プロセス配管・ユーティリティの支持構造と責任分界の策定**
  - 背景・理由: 特殊ガス配管を建築梁から吊る場合の荷重（地震力含む）を構造計算に算入し、建築・設備・プロセス業者の「誰が設計・施工・検査するか」を明確にする必要がある。

## 📚 必要知識・根拠法令 (Required Knowledge)
- **高圧ガス保安法（一般高圧ガス保安規則）**
  - 適用ポイント: 第一種・第二種製造者の区分確認、およびガス設備から境界線・火気までの法定離隔距離の算出と配置計画への適用。
- **労働安全衛生法（クレーン等安全規則）**
  - 適用ポイント: 天井クレーンの点検台、タラップ、歩廊の設置基準、および走行時の安全確保のための離隔距離規定。
- **建築基準法 第2条、第22条、第35条、令第112条**
  - 適用ポイント: 特殊建築物（工場）の用途規制、延焼ラインによる外壁性能、避難歩行距離、面積区画および危険物施設との防火区画。
- **酸素欠乏症等防止規則**
  - 適用ポイント: 不活性ガス（CO2等）を使用するエリアにおける換気設備（床面吸込）の設置基準と断面計画。
- **工場立地法・騒音規制法・振動規制法**
  - 適用ポイント: 生産設備の稼働に伴う近隣への影響評価、防音壁の設置要否、緑地面積率の算定。
- **消防法（危険物規制に関する政令）**
  - 適用ポイント: LPG等の指定数量の計算、および危険物貯蔵所としての構造・消火設備基準の適用。

## ⚠️ 重要調整・リスク事項 (Critical Coordination)
- **調整対象: [[構造設計者]]**
  - **想定リスク**: クレーン走行時の梁のたわみが防火シャッターや設備配管に干渉するリスク、および重量機械による床スラブのひび割れ。
  - **解決戦略 (PM/CM視点)**: クレーン走行荷重と変位制限を明記した構造計算書を確認し、意匠断面図で数センチ単位のクリアランス（バッファ）を確保した上で、3D干渉チェック図を作成し合意する。
- **調整対象: [[設備設計者（電気・機械）]]**
  - **想定リスク**: 防爆エリア内での非防爆機器の配置、およびガス検知器・連動遮断弁の設置場所と壁貫通スリーブの不整合。
  - **解決戦略 (PM/CM視点)**: 防爆エリア図を早期に提示し、電気設備プロット図と重ね合わせる。ガス漏れ検知時の連動フロー図を作成し、建築側の防火区画貫通処理と整合させる。
- **調整対象: [[施工部門（D&Bチーム）]]**
  - **想定リスク**: Sグレード鉄骨の製作リードタイム不足、および土間コンクリートの平坦度確保のための施工手順の欠如。
  - **解決戦略 (PM/CM視点)**: 特記仕様書に鉄骨精度（クレーン走行梁の取付精度）と床の平坦度基準を明記。確認済証交付時期と鉄骨発注時期をリンクさせた全体工程表で合意形成する。
- **調整対象: [[プロセス担当（ガス供給会社）]]**
  - **想定リスク**: ガス配管ルートが建築のブレースやクレーン走行空間と物理的に干渉し、着工後に配管ルートの変更を余儀なくされる。
  - **解決戦略 (PM/CM視点)**: プロセス配管図を意匠図にオーバーレイし、支持金物の位置と建築構造体への負荷を事前に検証。責任分界点（インターフェースリスト）をマトリクス形式で作成し、捺印合意する。

### 1. 主要な設計プロセスと決定事項

このフェーズでの意匠設計者の役割は、単なる図面作成ではなく、**「法規・機能・構造・設備」の整合性を図り、建築主事や指定確認検査機関と合意形成を行うこと**にあります。

*   **配置計画の最終確定**
    *   **延焼ラインの検討:** 法22条区域における外壁・軒裏の構造特定。
    *   **車両動線と消防活動空地:** 大型トラックの回転半径、非常用進入口（代替進入口）の確保。
    *   **高圧ガス設備との離隔距離:** 高圧ガス保安法に基づく、処理設備・貯蔵設備から境界線や火気までの法定離隔距離の確保。
*   **平面・断面計画のFIX**
    *   **防火区画の策定:** 工場用途のため、面積区画（1,500㎡以内等）や用途別区画の検討。特に特殊ガス使用エリアの防爆・防火措置。
    *   **避難計画:** 歩行距離の確認（重複歩行距離、重複避難経路）。
    *   **クレーン有効高さの確保:** 走行レール面からトラス下端までのクリアランス、およびフック有効高さの最終確認。
*   **仕様の決定（確認申請図書への反映）**
    *   **内装制限:** 工場・倉庫の制限緩和（法別表第1）の適用可否と、不燃材料の選定。
    *   **屋根・外壁:** 法22条区域に基づく不燃性能の担保。

### 2. 必要な法令知識

工場建築、かつ特殊ガスを扱うプロジェクトでは、建築基準法以外の「関連法規」との整合が最大の鍵となります。

*   **建築基準法**
    *   **第2条第2号（特殊建築物）:** 工場としての用途規制。
    *   **第22条・23条:** 屋根・外壁の不燃性能。
    *   **第35条（避難施設等）:** 避難規定。
    *   **令第112条（防火区画）:** 面積区画、および高圧ガス貯蔵庫等の危険物施設との区画。
*   **高圧ガス保安法（最重要）**
    *   CO2, LPG, O2の貯蔵量・処理量に基づく届出・許可申請。
    *   「第一種製造者」「第二種製造者」等の区分確認。
    *   建築確認申請に先立ち、または並行して都道府県の「高圧ガス担当課」との事前協議が必要。
*   **消防法**
    *   **危険物規制:** LPG等の貯蔵量による「指定数量」の計算。
    *   **消防用設備:** 自動火災報知設備、消火器、屋内消火栓、避難器具の選定。
*   **バリアフリー法・地方自治体条例**
    *   工場であっても、事務所併設の場合は努力義務や義務規定が発生するケースがある。
*   **都市計画法（開発許可）**
    *   敷地面積や立地により、法第29条の許可が必要な場合の工程管理。

### 3. 他分野との調整事項（コーディネート）

D&B方式では、意匠設計者がハブとなり、以下の専門分野を統合します。

*   **構造設計者との調整**
    *   **クレーン走行荷重:** 走行時の振動・衝撃荷重による柱・梁の変位制限（クレーンの走行性能に直結）。
    *   **杭基礎・構造スラブ:** 重量機械やクレーン荷重を考慮した床荷重（N/㎡）の設定。
    *   **Sグレード鉄骨:** 溶接品質や製作図承認プロセスの確認。
*   **設備設計者（機械・電気）との調整**
    *   **特殊ガス配管ルート:** ピット走行か天井走行か。ガス漏れ検知器の設置位置。
    *   **換気計画:** ガス漏れ時の強制換気、および工場全体の熱負荷計算。
    *   **受変電設備:** キュービクルの位置と、クレーン・生産設備への電力供給ルート。
*   **施工部門（D&Bチーム）との調整**
    *   **コスト管理:** 意匠仕様が予算内に収まっているかの確認。
    *   **工期:** 杭工事や鉄骨製作のリードタイムを考慮した確認済証交付時期の設定。
*   **PM/CM（発注者側）との調整**
    *   **生産ラインとの整合:** 建築側の柱位置が生産ラインの邪魔にならないか。
    *   **将来計画:** 増築や設備更新の可能性のヒアリング。

### 4. 本プロジェクト特有の留意点（シニアアーキテクトの視点）

1.  **高圧ガス申請と建築確認の「デッドヒート」**
    高圧ガスの許可が下りないと建築確認の済証が出せない（またはその逆）という「鶏と卵」の状態を避けるため、行政窓口との並行協議をリードします。特にLPGとO2は可燃性・助燃性ガスであるため、離隔距離の確保が配置図を左右します。
2.  **天井クレーンの「たわみ」と「クリアランス」**
    構造的な強度だけでなく、クレーン走行時の梁のたわみが、屋根裏の設備配管や防火シャッターに干渉しないよう、断面的な余裕（バッファ）を数センチ単位で管理します。
3.  **Sグレード鉄骨の品質管理**
    大規模工場では鉄骨精度が建物の寿命を左右します。確認申請図書に「国土交通大臣認定」等の仕様を明記し、施工段階での検査項目をこの時点で整理しておきます。
4.  **PM/CMへの説明責任**
    発注者側にPM/CMがいる場合、彼らは「なぜこの法規判断が必要なのか」「コストへの影響は」を論理的に説明することを求めます。法規の解釈（特に緩和規定の適用）については、根拠となる告示や技術基準を整理した「法規チェックリスト」を提示し、合意形成を迅速化させます。

### 2. PM/CMの批判的レビュー (Review)
<details><summary>クリックして展開</summary>

PM/CMの立場から、この初期案を厳しく査定させていただきます。

この案は「意匠設計者」としての体裁は整っていますが、**「工場建築」という特殊解に対する解像度が低すぎます。** このまま進めれば、実施設計中盤から工事段階にかけて、数千万〜数億円単位の追加コストや、数ヶ月単位の工期遅延を招く「地雷」がいくつも見受けられます。

特に「D&B（設計施工一括）方式」において、意匠設計者が「法規の整理」に終始し、**「プロセス（生産設備）と建築のインターフェース」**を疎かにすることは致命的です。

以下に、プロのPM/CMとして、徹底的に批判的な視点から「抜け漏れ」と「リスク」を指摘します。

---

### 1. 「特殊ガス（CO2, LPG, O2）」に関する認識の甘さとリスク

初期案では高圧ガス保安法への言及がありますが、建築実務への落とし込みが不十分です。

*   **防爆区画と電気設備のコンフリクト:**
    LPG（可燃性）を扱う場合、漏洩時の滞留を考慮した「防爆エリア」の設定が必要です。意匠設計者が防爆範囲を確定させない限り、設備設計者は「防爆仕様の照明・スイッチ・モーター」を選定できず、コストが跳ね上がります。
*   **酸素欠乏・中毒リスクへの配慮:**
    CO2は不活性ガスですが、漏洩すれば酸素欠乏症を引き起こします。ピット（溝）内にガスが滞留するリスクはないか？ 換気扇の吸込口を「床面付近」に設けるなどの断面計画が、意匠図に反映されていますか？
*   **ガス検知器と連動遮断弁の設置場所:**
    これらは「意匠図」には現れませんが、壁の貫通部やスリーブ、電気配線ルートに直結します。確認申請後に「ここに壁が必要だった」「この壁は貫通できない」という手戻りは許されません。

### 2. 「天井クレーン」と構造・設備の致命的な取り合い

「たわみ」への言及だけでは不十分です。

*   **クレーン給電線（トロリー線）のスペース確保:**
    クレーン走行レールの側方には、給電用のトロリー線や集電器のスペースが必要です。意匠設計者が「柱芯からクレーン端部までのクリアランス」を数センチ単位で把握していないと、柱にクレーンが激突するか、給電線が設置できなくなります。
*   **メンテナンス用タラップと歩廊（キャットウォーク）:**
    クレーンの点検には労働安全衛生法に基づく点検台やタラップが必要です。これが「確認申請図」に載っていないと、後付けで防火区画を貫通したり、避難経路を塞いだりするトラブルになります。
*   **走行レールの「直進度」と鉄骨精度:**
    S造平屋でスパンが広い場合、建方の精度がクレーンの脱輪に直結します。意匠設計者は、施工部門に対し「クレーン走行梁の取付精度」を特記仕様書で厳格に規定させる義務があります。

### 3. 「床（土間コンクリート）」という盲点

工場において「床」は生産設備の一部です。

*   **床の平坦度（レベル精度）:**
    クレーン下で精密機械を扱う場合、通常の建築基準（±10mm程度）では不十分なケースが多い。超平坦床（スーパーフラット）が必要か否か、このフェーズで確定させなければ、土間コンクリートの打設計画が狂います。
*   **床目地とフォークリフトの動線:**
    大型トラックやフォークリフトが頻繁に通る場所の「目地」は、床の崩壊を招く最大の原因です。意匠設計者は、車両動線に基づいた「目地割図」を作成し、構造設計者と配筋の補強を協議すべきです。

### 4. D&B方式における「責任分界点（インターフェース）」の欠如

*   **生産設備（プロセス）配管の支持:**
    特殊ガスの配管は誰が支持しますか？ 建築の梁から吊る場合、その荷重（および地震力）は構造計算に含まれていますか？ 「設備は別途」という逃げは、D&Bでは通用しません。
*   **ユーティリティの引き込み位置:**
    高圧ガス、電力、給排水の引き込み位置が、将来の増設計画や車両動線と干渉していないか。特に高圧ガス貯蔵庫の「火気との離隔」は、敷地境界だけでなく、隣接する「喫煙所」や「給湯室の排気口」も対象になります。

### 【追加すべき事項リスト】PM/CMからの要求

初期案に以下の項目を強制的に追加し、設計プロセスを再構築してください。

1.  **「防爆エリア図」の作成と電気設計への早期提示**
    *   LPG等の可燃性ガスの漏洩シミュレーションに基づく、防爆等級と範囲の確定。
2.  **「クレーン走行空間・メンテナンス空間」の3D干渉チェック**
    *   梁、柱、ブレース、照明器具、消火配管、クレーン本体の物理的干渉をセンチ単位で検証。
3.  **「高圧ガス保安法」に基づく離隔距離プロット図**
    *   確認申請図の配置図に、ガス設備からの法定離隔距離（第一種・第二種保安距離）を重ね書きし、将来の増設スペースを確保。
4.  **「床荷重・平坦度・目地」の要求仕様書（オーナーズ・リクワイアメント）の作成**
    *   生産設備の重量、フォークリフトの仕様に基づいた床のスペック確定。
5.  **「環境・公害対策」の事前協議記録**
    *   工場立地法、騒音規制法、振動規制法に基づく、近隣への影響評価と対策（防音壁の必要性など）。
6.  **「マトリクス状の責任分界表（インターフェースリスト）」**
    *   建築、構造、設備、プロセス（ガス供給会社）、施工の各者が「誰が何を設計し、誰が施工し、誰が検査するか」の明確化。

**PM/CMとしての結論：**
「意匠設計」を「建物の形を決めること」と考えているなら、このプロジェクトは失敗します。**「プロセスの容器」としての機能を完璧に定義すること**に注力してください。法規チェックリストは「最低限の宿題」に過ぎません。
