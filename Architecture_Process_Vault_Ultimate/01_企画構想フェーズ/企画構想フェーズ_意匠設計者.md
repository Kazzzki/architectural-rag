# [[企画・構想フェーズ]] における [[意匠設計者]] の役割と意思決定

## 🎯 主要な決定事項 (Primary Decisions)
- **特殊ガス（LPG/O2/CO2）の物理的特性に基づく配置・床レベル計画**
  - 背景・理由: LPG（空気より重い）の滞留による爆発リスクを回避するため、床レベルのピットや溝の原則廃止、または検知器連動の強制換気設備を決定する必要がある。また、O2（支燃性）供給エリア周辺での油脂を含むシール材・塗装の禁止（禁油処理）を仕様化し、初期段階でコストと材料選定に反映させるため。
- **高圧ガス保安法に基づく「障壁」の自立構造と建築動線の整合**
  - 背景・理由: 第一種・第二種保安距離が確保できない場合の「障壁（防護壁）」設置は、建築基準法上の「延焼のおそれのある部分」や避難動線とコンフリクトする。障壁の自立構造による基礎への影響と、消防法上の開口部制限を同時に解決する配置を確定させる必要がある。
- **クレーン走行レールの許容変位制限と外壁追従仕様の策定**
  - 背景・理由: S造のしなりによる柱の倒れや梁のたわみが、クレーンの脱輪や噛み合わせ不良を招くリスクがある。クレーンメーカーが要求する「走行レールの許容変位」を構造条件として固定し、意匠側で外壁パネルの目地幅や追従金物を設計に組み込む必要がある。
- **重荷重床の支持方式（構造スラブ vs 置き床）と不同沈下対策**
  - 背景・理由: 杭支持の柱と地盤支持の床の間で発生する不同沈下は、フォークリフトの走行性やクレーン精度に致命的な影響を与える。クレーン基礎と一般床の縁切り詳細、およびスライドジョイント等の段差防止策を決定し、FF値/FL値（平坦度）の具体的数値を設定するため。
- **メンテナンス用歩廊（キャットウォーク）および点検台の空間確保**
  - 背景・理由: 労働安全衛生法に基づくクレーン点検スペースや高所ガス配管の点検ルートは、建築面積や容積率に算入される。意匠・構造・設備で共有できるルートを早期策定し、法規上の最大床面積を確保するため。
- **将来の拡張・更新計画に伴う「ノックアウトパネル」の配置**
  - 背景・理由: 将来のクレーン増設（2.8t→5t等）や生産ライン変更時の搬入出ルートを想定し、構造的欠損を最小限に抑えつつ容易に解体・復旧可能な壁（ノックアウトパネル）の範囲を特定するため。

## 📚 必要知識・根拠法令 (Required Knowledge)
- **高圧ガス保安法（一般高圧ガス保安規則、液化石油ガス保安規則）**
  - 適用ポイント: 貯蔵量による事業所区分、保安距離の算定、障壁の構造基準、警戒標識の設置位置の特定。
- **建築基準法 第22条・23条（不燃化）、第27条（耐火構造）、第35条（避難）**
  - 適用ポイント: 工場の規模・用途に応じた耐火要求の確認、大空間における歩行距離・重複歩行距離の検証。
- **消防法（危険物製造所等に関する政令）**
  - 適用ポイント: 指定数量以上の危険物（油脂類等）を取り扱う場合の建築的制限、消火設備（移動式粉末消火設備等）の配置スペース確保。
- **労働安全衛生法（クレーン等安全規則）**
  - 適用ポイント: 吊上げ荷重3t以上のクレーン設置届、落成検査に必要な点検台・歩廊の設置基準、非常停止時のメンテナンススペース確保。
- **工場立地法**
  - 適用ポイント: 敷地面積9,000㎡以上または建築面積3,000㎡以上の場合の緑地率・環境施設面積の確保と届出時期の管理。
- **米国コンクリート学会 ACI 117（FF値/FL値）**
  - 適用ポイント: 超平坦床が求められる場合の床仕上げ精度の定量的設定と検査方法の規定。

## ⚠️ 重要調整・リスク事項 (Critical Coordination)
- **調整対象: [[構造設計者]]**
  - **想定リスク**: クレーンの動的負荷（加減速時の水平力）による建屋の揺れが、精密機器や外壁材に与える影響の過小評価。
  - **解決戦略 (PM/CM視点)**: クレーンメーカーから走行速度・衝撃荷重の数値を早期に取得し、構造計算書における「許容変位量」と意匠図の「目地詳細」を突合させた比較検討図で合意形成する。
- **調整対象: [[設備設計者（MEP）]]**
  - **想定リスク**: ガス漏洩検知器の信号と、建築側の排気ファン・防火ダンパーの連動制御フローの曖昧さによる、火災時の延焼リスク。
  - **解決戦略 (PM/CM視点)**: 「ガスハザードマップ」を作成し、検知器の配置、防爆エリアの範囲、連動する空調・換気設備の制御ロジックを記載したインターフェース図を作成・承認する。
- **調整対象: [[施工部門（DBチーム）]]**
  - **想定リスク**: Sグレード鉄骨の長納期化に伴う、詳細設計完了前の見切り発注による、後続の設備貫通穴等の追加工事コスト増。
  - **解決戦略 (PM/CM視点)**: 「重要図面（工作図）承認フロー」を確立し、鉄骨発注前に設備スリーブ・クレーンブラケット位置を確定させるための「先行図面パッケージ」をPM/CMに提示する。
- **調整対象: [[行政機関（消防・高圧ガス・労基署）]]**
  - **想定リスク**: 消防法（開口部設置要求）と高圧ガス保安法（遮蔽要求）の矛盾による、確認済証交付の遅延。
  - **解決戦略 (PM/CM視点)**: 三者協議のログを管理し、矛盾点を抽出した上で、防護壁の配置や防火設備の仕様について代替案（性能評価等）を用いた同時並行協議を行う。
- **調整対象: [[PM/CM（発注者側）]]**
  - **想定リスク**: 建築工事と生産設備工事の責任境界（バッテリーリミット）の不明確化による、二次側配管・配線の施工漏れや二重計上。
  - **解決戦略 (PM/CM視点)**: 「インターフェース・リスト（区分表）」を作成し、ガス供給元から使用点までの配管支持架台、電源供給点、信号取合位置をBIM上で可視化して合意する。

意匠設計者は「建物の器」を作るだけでなく、生産プロセスを理解し、法規と技術の整合性を取る「オーケストレーター」としての役割を担います。

2.  **ボリュームスタディと配置計画**:

### 2. 主要な決定事項

このフェーズで確定させるべき「動かせない軸」は以下の通りです。

*   **モジュール（柱スパン）の決定**:
    *   クレーンの走行レールスパンと、S造の経済スパン（一般に10〜12m程度、あるいは大スパン）の整合。
*   **床レベルと構造スラブ仕様**:
    *   重荷重・動荷重に耐える構造スラブの厚みと仕上げ（防塵塗装、耐衝撃性）。
    *   フォークリフトの走行性を考慮した不陸精度の設定。
*   **特殊ガス貯蔵所の位置**:
    *   LPG、O2等の高圧ガス保安法に基づく「第一種・第二種保安距離」の確保。
    *   建物本体からの離隔距離と、延焼ラインの検討。
*   **BCP・安全性**:
    *   ガス漏洩時の換気計画（自然換気 vs 強制換気）の基本方針。
    *   クレーン非常停止時のメンテナンススペース確保。

### 3. 必要な法令知識

工場設計において意匠設計者が精通しておくべき法規は、建築基準法に留まりません。

*   **建築基準法**:
    *   **第22条・23条**: 屋根・外壁の制限（不燃材料等）。
    *   **第27条（耐火構造等）**: 特殊建築物（工場）の規模による耐火要求の有無（平屋であれば緩和が多いが、面積により注意）。
    *   **第35条（避難）**: 歩行距離、重複歩行距離の確認（大空間工場では特に重要）。
*   **高圧ガス保安法**:
    *   CO2, LPG, O2の貯蔵量による「事業所」の区分（一般高圧ガス、液化石油ガス）。
    *   警戒標識の設置位置、障壁（防護壁）の要否。
*   **消防法**:
    *   指定数量以上の危険物を取り扱う場合の「危険物製造所等」への該当性。
    *   自動火災報知設備、消火設備（スプリンクラー、移動式粉末消火設備等）の選定。
*   **工場立地法**:
    *   敷地面積が一定以上（一般に9,000㎡以上等）の場合の緑地率確保。
*   **労働安全衛生法**:
    *   クレーン設置届（吊上げ荷重3t以上の場合、落成検査が必要）。

### 4. 他分野との調整事項（インターフェース管理）

設計施工一括（DB）方式では、早期に施工・設備サイドと合意形成を図ることがコスト確定の鍵となります。

*   **構造設計者との調整**:
    *   **クレーン反力**: 走行荷重、衝撃荷重による柱・梁のサイズへの影響。
    *   **杭基礎**: 構造スラブと基礎の縁切り、あるいは一体化の判断。
    *   **Sグレード鉄骨**: 製作期間の長いSグレード材の早期発注に向けた図面確定時期の調整。
*   **設備設計者（MEP）との調整**:
    *   **ガス配管ルート**: 意匠的な露出の許容範囲と、メンテナンス用キャットウォークの共有。
    *   **電気容量**: クレーンおよび生産設備の電力負荷確認と、受変電設備（キュービクル）の配置。
*   **施工部門（DBチーム）との調整**:
    *   **揚重計画**: 大スパン梁の地組スペースや、クレーンレールの施工精度確保。
    *   **コスト管理**: PM/CMに対するコスト妥当性の説明資料作成。
*   **PM/CM（発注者側）との調整**:
    *   **生産設備との境界（BIM/CIM活用）**: 発注者が手配する生産ラインと、建築工事の取り合い（インターフェース）の明確化。

### 1. 特殊ガス（CO2, LPG, O2）に関する致命的な認識不足

初期案では「法規の確認」に留まっているが、意匠設計者が踏み込むべきは**「物理的特性への対応」**だ。

*   **LPG（空気より重い）の滞留リスク**:
    *   **指摘**: LPGは床面に滞留する。床レベルにピットや溝、あるいは「床下空間」がある場合、そこが爆発限界に達するリスクを無視している。
    *   **追加事項**: ピットの廃止、あるいはピット内強制換気・検知器連動の検討。床仕上げの不燃化だけでなく、静電気防止仕様の要否。
*   **O2（支燃性ガス）の危険性**:
    *   **指摘**: 酸素濃度上昇による火災リスク。周囲の建材に油脂分が含まれる材料（シール材、塗装等）を使用することの危険性。
    *   **追加事項**: ガス供給エリア周辺の材料選定（禁油処理、特定材料の排除）。
*   **高圧ガス保安法の「障壁」と「延焼ライン」のコンフリクト**:
    *   **指摘**: 保安距離が取れない場合の「障壁（防護壁）」の設置は、構造荷重や避難動線に多大な影響を与える。
    *   **追加事項**: 障壁の自立構造検討と、建築基準法上の「延焼のおそれのある部分」との整合。

### 2. クレーンと建屋の「動的インターフェース」の甘さ

「揚程から高さを決める」のは初歩だ。PMが恐れるのは**「クレーンが動かない」「外壁が割れる」**ことだ。

*   **走行レールの変位制限（クリアランス）**:
    *   **指摘**: S造はしなる。クレーン走行時の柱の倒れや梁のたわみが、クレーンの脱輪や噛み合わせ不良を引き起こす。
    *   **追加事項**: クレーンメーカーが要求する「走行レールの許容変位」を構造設計者に突きつけ、意匠側で外壁パネルの目地（追従性）を設計すること。
*   **クレーン給電方式と建築の取り合い**:
    *   **指摘**: 絶縁トロリー等の給電ラインの支持方法。意匠図に反映されていないと、後付けの支持部材が意匠や他設備と干渉する。
    *   **追加事項**: 給電ラインの支持スパンと、鉄骨側ブラケットの先行計画。

### 3. 「杭基礎」と「重荷重床」の沈下リスク（不同沈下）

*   **指摘**: 柱は杭で支持されるが、床（土間コンクリート）は「置き床（地盤支持）」なのか「構造スラブ（杭支持）」なのか？
    *   初期案では「構造スラブ」とあるが、全域を杭支持にするのか？ もし一部が地盤支持なら、柱（杭）と床（地盤）の間で必ず**不同沈下**が起きる。
*   **追加事項**: クレーン基礎と一般床の縁切り詳細。フォークリフトが通過する際の段差防止策（スライドジョイント等）。

### 4. DB（設計施工一括）方式における「責任境界（バッテリーリミット）」の曖昧さ

DB方式で最も揉めるのは、**「誰がどこまで工事し、誰が性能保証するか」**だ。

*   **生産設備（プロセス）との境界**:
    *   **指摘**: ガスの配管は「建築設備」か「プロセス設備」か？ 供給元（ボンベ庫）から使用ポイントまでの「二次側配管」のルートと支持架台の負担区分が不明。
    *   **追加事項**: インターフェース・リスト（区分表）の作成。特に「ガス検知器の信号をどこの盤で受け、どのファンを回すか」という制御フローの確定。
*   **Sグレード鉄骨の調達リスク**:
    *   **指摘**: 「早期発注」と書いているが、DBでは施工側が勝手に発注を進めるリスクがある。
    *   **追加事項**: 意匠設計者として、発注前に「承認すべき重要図面（工作図）」の特定と、PM/CMへの報告ルートの確立。

### 5. 労働安全衛生法とメンテナンス性の欠如

*   **指摘**: クレーンや高所のガス配管の「点検」をどう行うか？
*   **追加事項**: 
    *   **点検用歩廊（キャットウォーク）**: 意匠・構造・設備で共有できるルートの策定。
    *   **クレーン点検台**: 法令で定められた点検スペースの確保（建築面積に算入されるため、初期に確定が必要）。

### PM/CMとして追加を命じる「具体的アクションリスト」

1.  **ガスハザードマップの作成**: 
    *   ガスの種類ごとに「漏洩時の滞留シミュレーション」を想定し、換気口・検知器・防爆エリアをプロットした図面を作成せよ。
2.  **クレーン仕様書（建築要求版）の早期確定**: 
    *   揚程だけでなく、走行速度、加減速時の水平力、レールの許容変位をメーカーから引き出し、構造設計者と合意せよ。
3.  **床レベル精度の数値化**: 
    *   「不陸精度の設定」という曖昧な表現はやめ、平坦度（FF値/FL値）の具体的数値を設定し、施工側とコスト影響を協議せよ。
4.  **行政事前協議のログ管理**: 
    *   高圧ガス保安法、消防法（危険物）、労働基準監督署（クレーン設置届）の三者協議を同時並行で進め、それぞれの回答の矛盾（例：消防は開口部を求めるが、高圧ガスは遮蔽を求める等）を早期に抽出せよ。
5.  **将来計画の「空間的予約」**: 
    *   将来の拡張や設備更新時、どの壁を壊し、どのルートで搬入するか。その際の「構造的欠損」をあらかじめ織り込んだ計画（ノックアウトパネル等）を提示せよ。

**「綺麗にまとめる」のは不要だ。このプロジェクトを殺さないための「泥臭い整合」に注力せよ。**
